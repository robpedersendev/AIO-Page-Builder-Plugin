Structured page building with AI-assisted planning. Private distribution; not submitted to the WordPress.org plugin directory.

== Description ==

AIO Page Builder helps teams plan, build, and maintain WordPress pages using structured templates, execution workflows, and optional AI providers. Built pages and export data are designed to remain usable if the plugin is deactivated or removed, subject to product configuration and uninstall options.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or install via your approved distribution channel.
2. Activate the plugin through the Plugins screen in WordPress.
3. Configure capabilities and AI provider settings per your site policy.
